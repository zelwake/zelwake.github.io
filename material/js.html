String manipulation

.split() separate string into substring based on separator put in (). Result is array without separator
.slice(startIndex, endIndex) extracts a section of a string and returns it as a new string, without modifying the original string. endIndex is optional
.toLowerCase .toUpperCase convert text to either lower case or upper case
.indexOf(searchString, position) searches string and return first occurence of specified symbol or string. Position is optional

Array manipulation

.push() add one element to end
.unishft() add one element to start
.pop() remove one element from end
.shift() remove one element from start
.splice(startIndex, numberOfItems, newlyAddedItems) modifies multiple elements next to each other
.slice(startIndex, endIndex) copy selected range, startIndex and endIndex are optional, undefined () start at 0
... whole array (...thisArray)
.indexOf() checks if value exists, return position, index or -1
.join() turn array into string, using separator of choice put inside ()
.sort() sort array alphabetically, for numerical: .sort(a, b) { return a - b }

Object manipulation

object.item = value add item to object with value or modifies value of existing item
delete object.item removes item from object
.hasOwnProperty() / property in object checks if property exists, returns true of false
for (item in object) function goes through each item in object
Object.keys() function return array of properties in object
