Array manipulation

.push() add one element to end
.unishft() add one element to start
.pop() remove one element from end
.shift() remove one element from start
.splice(startIndex, numberOfItems, newlyAddedItems) modifies multiple elements next to each other
.slice(startIndex, endIndex) copy selected range
... whole array (...thisArray)
.indexOf() checks if value exists, return position, index or -1

Object manipulation

object.item = value add item to object with value or modifies value of existing item
delete object.item removes item from object
.hasOwnProperty() / property in object checks if property exists, returns true of false
for (item in object) function goes through each item in object
Object.keys() function return array of properties in object
